# 季风 Monsoon - 个人风格分析抖音小程序

## 项目简介

季风 Monsoon 是一款基于大模型能力的抖音小程序，通过心理学问卷和照片分析，为用户提供个性化的色彩分析和风格建议。采用米白色底+黑色线条的优雅简约手绘风格设计。

## 功能特性

- 🎨 **智能风格分析**：基于问卷和照片的个性化分析
- 📸 **多角度照片上传**：手腕、脖子、正脸、半身、全身
- 🎯 **精准色彩推荐**：季节色彩理论 + 个性化色板
- 👗 **风格建议**：主风格、关键词、场景推荐
- 📋 **材质推荐**：适合的面料和材质建议
- 💬 **追问改进**：支持用户继续提问微调建议
- 📊 **历史记录**：本地存储分析报告
- 📈 **数据埋点**：用户行为统计和分析

## 技术架构

- **前端**：抖音小程序原生（TTML/TTSS/JS）
- **AI模型**：豆包1.5 thinking vision pro（视觉）+ 豆包1.5 pro 32k（文本）
- **存储**：本地缓存 + 云存储（图片上传）
- **配置化**：JSON配置文件，便于扩展

## 使用流程

### 用户使用流程

1. **引导页**：3页引导介绍
2. **主页**：选择"开始风格分析"
3. **问卷**：回答8道风格偏好问题
4. **照片上传**：上传5类照片（最少3张）
5. **生成中**：AI分析处理
6. **报告展示**：查看个性化风格报告
7. **追问改进**：继续提问微调建议
8. **历史记录**：查看历史分析报告

### 开发扩展

#### 修改问卷

编辑 `config/questionnaire.json`：

```json
{
  "questions": [
    {
      "id": "q1",
      "type": "multi_select",
      "title": "你的问题",
      "options": [...],
      "dimension": "stylePreference",
      "weight": 0.3
    }
  ]
}
```

#### 调整提示词

编辑 `config/prompts.json`：

```json
{
  "styleAnalysis": {
    "mainPrompt": "你的自定义提示词"
  }
}
```

#### 修改UI样式

编辑 `app.ttss` 中的CSS变量：

```css
page {
  --primary-bg: #F5F5DC;
  --primary-text: #333333;
  --accent-color: #D4A574;
}
```

## 核心功能

### 1. 问卷分析

- 8个维度：风格偏好、场合适应性、功能导向等
- 权重计算：不同维度的重要性权重
- 量化评分：1-4分制评分系统

### 2. 照片分析

- 肤色分析：色调、明度、饱和度
- 静脉颜色：判断冷暖色调
- 面部特征：自然、温暖、冷峻等

### 3. 风格报告

- **主色调**：季节色彩类型
- **推荐色板**：4个主色 + 6个扩展色
- **适合风格**：主风格、关键词、场景
- **推荐材质**：面料建议和说明
- **避免项**：不适合的颜色和搭配

### 4. 三个弹窗

- **推荐色板弹窗**：详细颜色使用建议
- **适合风格弹窗**：风格分析和搭配建议
- **推荐材质弹窗**：材质选择和护理建议
